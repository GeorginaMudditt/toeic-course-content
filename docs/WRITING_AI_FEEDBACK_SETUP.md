# TOEIC Writing — AI Feedback Setup

Students can click **Get AI Feedback** on E-mail 1, E-mail 2, and the Essay in **Introduction to TOEICⓇ Writing**. Feedback is generated by AI (not human marking).

## What you need to do

### 1. Get a free Gemini API key

1. Go to [Google AI Studio](https://aistudio.google.com/apikey)
2. Sign in with a Google account
3. Click **Create API key**
4. Copy the key (starts with `AIza...`)

The free tier is usually enough for classroom use. Limits apply per minute/day.

### 2. Add the key locally (for testing)

In the project root, open or create `.env.local` and add:

```env
GEMINI_API_KEY=your_api_key_here
```

Optional — change the model (default is `gemini-2.0-flash`):

```env
GEMINI_MODEL=gemini-2.0-flash
```

Restart the dev server after saving:

```bash
npm run dev
```

### 3. Add the key on Netlify (for the live site)

1. [Netlify](https://app.netlify.com) → your site → **Site configuration** → **Environment variables**
2. **Add variable**
   - Key: `GEMINI_API_KEY`
   - Value: your API key
3. **Save**
4. **Deploys** → **Trigger deploy** → **Deploy site** (required so the new variable is picked up)

Also add `GEMINI_API_KEY` to any preview/deploy contexts you use.

### 4. Push the code & update the resource in Supabase

The app code (API route + buttons) must be deployed, and the worksheet HTML must include `data-grammar-ai-feedback` on the three writing areas.

From the project folder:

```bash
git pull
npm run update:resource:supabase "Introduction to TOEICⓇ Writing" "introduction-toeic-writing-html.html"
```

(Commit and push first if those changes are only on your machine.)

### 5. Test as a student

1. Open the Writing introduction assignment
2. Type a short answer in **E-mail 1**
3. Click **Save**, then **Get AI Feedback**
4. You should see **Instant checks** (word count, task hints) and **AI feedback** below

If you see *"AI feedback is not configured yet"*, the API key is missing on the server — check Netlify env vars and redeploy.

## How it works

| Step | What happens |
|------|----------------|
| **Instant checks** | Runs on your server — word count, greeting/closing, task hints (free, no AI) |
| **AI feedback** | Sent to Google Gemini with the task prompt + student text — returns structured comments |
| **Save** | Stores answers and feedback in the student’s progress (Supabase) so they can return later |

- One feedback request per task per **60 seconds** (stops accidental spam)
- Label **Get AI Feedback** makes clear this is not human marking

## Privacy note for students

You may want to mention in class: their writing is sent to Google’s API to generate feedback. See [Gemini API terms](https://ai.google.dev/gemini-api/terms) for how Google handles API data on the free tier.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| "AI feedback is not configured yet" | Add `GEMINI_API_KEY` to Netlify and redeploy |
| "Please wait X seconds" | Normal rate limit; wait and try again |
| "AI feedback service is temporarily unavailable" | Gemini outage or quota — try later or check [AI Studio usage](https://aistudio.google.com/) |
| Button missing | Resource HTML needs `data-grammar-ai-feedback="email1"` etc.; redeploy app + update Supabase resource |
