Conditionals : Third, Mixed, and "If" Alternatives

Grammar reference and practice

Introduction

In this lesson, we'll explore more advanced conditional structures. These will help you discuss past situations, regrets, hypothetical scenarios across different time frames, and express conditions in more sophisticated ways using alternatives to "if."

Grammar Point #1: Third Conditional

Use: For unreal situations in the past; talking about things that didn't happen or regrets, and imagining different outcomes.

Structure:

  • If + Past Perfect, Would have + Past Participle (OR)
  • Would have + Past Participle + If + Past Perfect (no comma!)

Examples:

  • If I had known about the meeting, I would have attended. / I would have attended if I had known about the meeting.
  • If she had applied for the position, I'm sure she would have got it. / I'm sure she would have got the position if she had applied.
  • If we had responded faster, we wouldn't have lost the client. / We wouldn't have lost the client if we had responded faster.

Notes:

  • Contractions are commonly used (e.g. If I'd known about the meeting, I would've attended).
  • The past participle of "get" is "got" in UK English and "gotten" in US English.

Grammar Point #2: Mixed Conditionals

Use: Mixed conditionals are used when the time in the "if" clause and the time in the main clause are different. This allows us to connect past actions with present results, or present situations with past consequences.

Type 1: Past condition → Present result

Structure:

  • If + Past Perfect, Would + Infinitive (without "to") (OR)
  • Would + Infinitive (without "to") + If + Past Perfect (no comma!)

Examples:

  • If I had studied medicine, I would be a doctor now. / I would be a doctor now if I had studied medicine.
  • If we had hired more staff last year, we wouldn't be so busy now. / We wouldn't be so busy now if we had hired more staff last year.

Type 2: Present condition → Past result

Structure:

  • If + Past Simple, Would have + Past Participle (OR)
  • Would have + Past Participle + If + Past Simple (no comma!)

Examples:

  • If she was/were more organised, she wouldn't have missed the deadline. / She wouldn't have missed the deadline if she was/were more organised.
  • If I didn't work here, I wouldn't have met you. / I wouldn't have met you if I didn't work here.

Vocabulary #1: Alternatives to "If"

There are several other words and phrases we can use instead of "if" to express conditions. These often sound more formal or sophisticated in professional contexts.

Unless (= if not)

  • Unless we receive payment, we will suspend the service. / We will suspend the service unless we receive payment.
  • She won't get the position unless she improves her performance. / Unless she improves her performance, she won't get the position.

Provided (that) / Providing (that) (= only if, on the condition that)

  • You can work remotely provided that you meet all deadlines. / Provided that you meet all deadlines, you can work remotely.
  • We'll approve the budget providing the projections are accurate. / Providing the projections are accurate, we'll approve the budget.

As long as / So long as (= only if)

  • As long as you submit the report by Friday, there's no problem. / There's no problem as long as you submit the report by Friday.
  • You can take the afternoon off so long as your work is finished. / So long as your work is finished, you can take the afternoon off.

Assuming (that) / Supposing (that) (= if we accept that something is true)

  • Assuming the shipment arrives on time, we'll start production on Monday. / We'll start production on Monday assuming the shipment arrives on time.
  • Supposing we don't get the funding, what's our backup plan? / What's our backup plan supposing we don't get the funding?

On condition that (= only if; very formal)

  • The investor will fund the project on condition that we meet quarterly targets. / On condition that we meet quarterly targets, the investor will fund the project.

Otherwise (= if not; used after a statement)

  • We need to act quickly; otherwise, we'll lose the opportunity.
  • Submit your timesheet by Monday, otherwise you won't be paid on time.

In case (= because something might happen)

  • Take an umbrella in case it rains.
  • We have backup generators in case of power failures.

  1. If Christopher had prepared better, his presentation would have been more successful.
  2. The product launch wouldn't have failed if the marketing team had done more research.
  3. If Elena had negotiated harder, she would have got/gotten a better salary.
  4. We would have hired more staff if we hadn't spent so much money on the new premises last year.
  5. If the pandemic hadn't happened, the restaurant would still be operating today.
  6. The company would have avoided bankruptcy if management had made different decisions.

Practice #2: Mixed Conditionals

  1. If Ryan had accepted the job offer in 2020, he would live / would be living in Singapore now. Type: Past condition → Present result
  2. If Lauren were more confident, she would have applied for the management position last month. Type: Present condition → Past result
  3. If the company hadn't expanded into Asia, we wouldn't have these opportunities today. Type: Past condition → Present result
  4. If I spoke Mandarin, I would have got/gotten that job at the Shanghai office last year. Type: Present condition → Past result
  5. The startup would still be in business if the founders hadn't disagreed about the direction. Type: Past condition → Present result

  2. You can have flexible hours as long as you complete all your tasks.
  3. Assuming (that) the data is correct, we should proceed with the launch.

  1. Project debrief: "If the team had communicated more effectively during the project, we wouldn't have encountered so many problems. Moving forward, we'll succeed as long as everyone attends the weekly meetings."

  3. Risk assessment: "Supposing the market crashed unexpectedly, what contingency plans do we have in place?"
  4. Regret: "If I had listened to Pablo's advice last year, the department would be in a much better position today."
  5. Warning: "You need to improve your attendance; otherwise, we'll have to begin disciplinary procedures."
