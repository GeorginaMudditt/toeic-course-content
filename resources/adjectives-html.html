Grammar Focus #2: Order of Multiple Adjectives

In English, adjectives follow a specific order (unlike French). Use the memory trick OSASCOMP:

  1. Opinion (e.g., beautiful, ugly, nice, horrible)
  2. Size (e.g., big, small, tall, short)
  3. Age (e.g., old, young, new, ancient)
  4. Shape (e.g., round, square, flat)
  5. Colour (e.g., red, blue, green)
  6. Origin (e.g., French, American, Italian)
  7. Material (e.g., wooden, metal, plastic, cotton)
  8. Purpose (e.g., sleeping bag, running shoes)
üí¨ Examples:
  • A beautiful, large, old, round, wooden table
    (opinion + size + age + shape + material)
  • An expensive new Italian leather jacket
    (opinion + age + origin + material)
