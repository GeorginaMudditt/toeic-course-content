Conditionals: Zero, First, Second

What are conditionals?

Conditionals are sentences that express a condition and a result. They help us talk about real situations, possibilities, and imaginary scenarios. In this lesson, we'll learn about three types of conditionals.

Grammar Point #1: Zero Conditional

Use: For facts, general truths, and things that are always true.

Structure:

  • - If/When + Present Simple, Present Simple (OR)
  • - Present Simple + if/when + Present Simple (no comma!)

Examples:

  • - When employees work overtime, they receive extra pay. / Employees receive extra pay when they work overtime.
  • - If the server crashes, we lose data. / We lose data if the server crashes.
  • - If a customer complains, we offer a refund. / We offer a refund if a customer complains.

Grammar Point #2 : First Conditional

Use: For real and possible situations in the future.

Structure:

  • - If + Present Simple, Will + Infinitive (without "to") (OR)
  • - Will + Infinitive (without "to") + if + Present Simple (no comma!)

Note: "If" can only be replaced by "when" in sentences using the Zero Conditional.

Examples:

  • - If we finish early, we will go bowling. / We will go bowling if we finish early.
  • - If the client agrees, we will sign the contract tomorrow. / We will sign the contract tomorrow if the client agrees.
  • - If there are any problems, I will call you. / I will call you if there are any problems.

Grammar Point #3: Second Conditional

Use: For unreal, imaginary, or unlikely situations in the present or future.

Structure:

  • - If + Past Simple, Would + Infinitive (without "to") (OR)
  • - Would + Infinitive (without "to") + If + Past Simple (no comma!)

Note: "If" can only be replaced by "when" in sentences using the Zero Conditional.

Examples:

  • - If we had a bigger budget, we would hire more people. / We would hire more people if we had a bigger budget.
  • - If I were the CEO, I would change the working hours. / I would change the working hours if I were the CEO.
  • - If our office had a gym, employees would be healthier. / Employees would be healthier if our office had a gym.

Note: We often use "were" instead of "was" in second conditionals, especially in formal contexts, but both are grammatically correct.

  • - If I were you, I would accept the offer. (common) / If I was you, I would accept the offer. (less common)
  • - If he were more experienced, he would get the position. (formal) / If he was more experienced, he would get the position. (less formal)

Practice #4 : Mixed Conditionals

Choose the correct conditional type (Zero, First, or Second):

  1. If you ____________________ (add) sugar to coffee, it ____________________ (taste) sweeter. (Zero/First/Second)
  2. If the presentation ____________________ (go) well tomorrow, we ____________________ (win) the client. (Zero/First/Second)
  3. If I ____________________ (be) the manager, I ____________________ (change) the policy. (Zero/First/Second)
  4. When employees ____________________ (feel) valued, they ____________________ (perform) better. (Zero/First/Second)
  5. If this supplier ____________________ (deliver) late again, we ____________________ (find) a new one. (Zero/First/Second)
  6. Dmitri has applied for a promotion. He ____________________ (move) to the head office in Amsterdam if they ____________________ (offer) him the position. (Zero/First/Second)
  7. If you ____________________ (heat) metal, it ____________________ (expand). (Zero/First/Second)
  8. If the economy ____________________ (improve), more businesses ____________________ (invest) in technology. (Zero/First/Second)

Practice #5 : Real-World Application

Practice #1 : Zero Conditional

  1. If the printer runs out of ink, it stops working.
  2. When Aisha finishes a project, she sends a report to her supervisor.
  3. If the restaurant is full, customers wait in the lobby.
  4. When temperatures drop below zero, the roads become dangerous.
  5. If you don't save your document before you close it, you lose everything.
  6. When the alarm sounds, everyone leaves the building immediately.

Practice #2 : First Conditional

  1. If the meeting finishes on time, I will catch the 6 p.m. train.
  2. The project will fail if we don't work as a team.
  3. If Marcus sends the invoice today, the client will pay by Friday.
  4. We will need more staff if the business continues to grow.
  5. If the investors agree, we will launch the product next month.
  6. The caf√© will close if it doesn't attract more customers.

Practice #3 : Second Conditional

  1. If the company offered remote work, more people would apply for jobs.
  2. Jenny would be happier if she worked fewer hours.
  3. If we had better equipment, we would produce higher quality products.
  4. The restaurant would make more profit if it opened on Sundays.
  5. If Hassan spoke French, he would work at the Paris office.
  6. I wouldn't stay in this job if I had another option.

Practice #4 : Mixed Conditionals

  1. If you add sugar to coffee, it tastes sweeter. (Zero)
  2. If the presentation goes well tomorrow, we will win the client. (First)
  3. If I was/were the manager, I would change the policy. (Second)
  4. When employees feel valued, they perform better. (Zero)
  5. If this supplier delivers late again, we will find a new one. (First)
  6. Dmitri has applied for a promotion. He will move to the head office in Amsterdam if they offer him the position. (First)
  7. If you heat metal, it expands. (Zero)
  8. If the economy improves, more businesses will invest in technology. OR If the economy improved, more businesses would invest in technology. (First or Second - it depends on how likely you think it is)

Practice #5 : Real-World Application

  1. Email from manager: "If you need any help with the report, I will be in the office all afternoon."
  2. Team discussion: "This budget is so low. If we had a bigger budget, we could/would hire a specialist for this project."
  3. Training session: "When you receive a customer complaint, you follow the procedure in the handbook."
  4. Performance review: "If you meet your targets this year, you will receive a bonus."
  5. Break room conversation: "If I were you, I would talk to HR about the situation."
